# `vc-space`

> 设置组件之间的间距，类似`antdv`的`space`组件。

## 开发说明

之所以开发这个组件，是因为在使用`antdv`的`space`组件过程中发现一个很严重的问题，就是`space`组件会给每个子元素加一个外层元素`space-item`，当其中一个子元素删除之后（如`v-if`），其他元素的组件会发生重新加载！而这并不是我们想要的，在一些情况下会导致难以发现的问题，所以不再使用该组件，而重新开发了这个无副作用的组件替代。

## 配置项

| 属性        | 类型                 | 默认值       | 描述                                                                  |
| ----------- | -------------------- | ------------ | --------------------------------------------------------------------- |
| `block`     | `Boolean`            | 无           | 块级展示                                                              |
| `align`     | `String`             | 无           | 对齐方式，`start`、`end`、`center`、`baseline`                        |
| `direction` | `String`             | `horizontal` | 间距方向，`vertical`、`horizontal`                                    |
| `size`      | `String` \| `Number` | `small`      | 间距大小，`small`(`8px`)、`middle`(`16px`)、`large`(`24px`)或具体数值 |
